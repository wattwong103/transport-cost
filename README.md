# transport-cost
Bus - rail network

## Data sources

Transit station coordinates are sourced from OpenStreetMap via the Overpass API.
See `scripts/fetch-osm-stations.py` and `scripts/match-stations.py` for the
ingestion and reconciliation pipeline.
